# 🎓 Student Grade Calculator in Java

> Developed as part of my Java Development and Problem-Solving Journey under Decodelabs Internship.

## 📌 Project Overview
The Student Grade Calculator is a console-based Java application that accepts marks for multiple subjects, calculates the total marks and average percentage, assigns grades according to predefined grading criteria, and presents the student's academic performance in a structured format.

The application implements robust input validation and exception handling to ensure reliable and accurate result generation.

---

## ✨ Features
- Accepts marks for multiple subjects
- Validates marks within the range of 0–100
- Calculates total marks
- Calculates average percentage
- Assigns grades automatically
- Handles invalid inputs using exception handling
- Displays results in a clear and structured format
