# 🎯 Number Guessing Game in Java

> Developed as part of my Java Development and Problem-Solving Journey under Decodelabs Internship.

## 📌 Project Overview
The Number Guessing Game is a console-based Java application that generates a random number between 1 and 100 and challenges the user to guess it. The application provides real-time feedback by indicating whether the guess is too high or too low until the correct number is guessed.

The project also incorporates features such as limited attempts, multiple rounds, score tracking, and exception handling to provide an interactive and robust user experience.

---

## ✨ Features
- Generates a random number between 1 and 100
- Accepts user guesses through console input
- Provides feedback:
  - Too High
  - Too Low
  - Correct Guess
- Supports limited attempts
- Allows multiple rounds of gameplay
- Tracks final scores
- Handles invalid inputs using exception handling
- Maintains a live loop until the logical condition is satisfied
